# Roadmap

Ideas and future improvements for this collection.

## Short Term
- Add more frontend examples to remaining dApp-style repos
- Increase test coverage across more repositories
- Add more GitHub Actions CI workflows

## Medium Term
- Create video walkthroughs or tutorials for popular starters
- Add example full-stack applications that combine multiple starters
- Improve cross-repo consistency (e.g., shared utilities)

## Long Term
- Expand to more chains (e.g., additional L2s or new ecosystems)
- Build a template generator or CLI tool to scaffold new starters
- Grow the community around these starters

Contributions and suggestions are very welcome!
