# Troubleshooting

Common issues and solutions when using these starters.

## Deployment Issues
- **Error: insufficient funds** — Make sure your wallet has enough native token for gas on the target network.
- **Contract verification fails** — Double-check the constructor arguments and network settings.

## Frontend Issues
- **Wallet not connecting** — Ensure you're using a supported wallet and the correct network.
- **Transaction fails** — Check for correct contract addresses and sufficient token balances.

## Cross-Chain Issues
- **Bridge transaction stuck** — Check the bridge explorer (e.g., for CCTP) and wait for attestations.
- **Wrong chain** — Always verify you're on the correct testnet or mainnet.

## General
- Check the individual repo's README and INTEGRATION_GUIDE.md for specific guidance.
- Feel free to open an issue if you encounter a problem not covered here.
