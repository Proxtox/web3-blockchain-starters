# Web3 & Blockchain Starters Showcase

A curated collection of high-quality starter repositories for **Base, Arc, Solana, Ethereum, Sui, and BSC**.

## Cross-Chain & Interoperability
- [base-arc-cctp-bridge](https://github.com/Proxtox/base-arc-cctp-bridge) — Seamless USDC bridging between Base and Arc using CCTP
- [ethereum-arc-interoperability-examples](https://github.com/Proxtox/ethereum-arc-interoperability-examples) — Cross-chain messaging and data sharing
- [arc-cross-chain-liquidity-router](https://github.com/Proxtox/arc-cross-chain-liquidity-router) — Liquidity routing across Arc + connected chains
- [multi-chain-stablecoin-toolkit](https://github.com/Proxtox/multi-chain-stablecoin-toolkit) — Reusable tools & SDKs for stablecoins across 6 chains

## DeFi & Lending
- [arc-stablecoin-lending-pools](https://github.com/Proxtox/arc-stablecoin-lending-pools) — Isolated stablecoin lending pools on Arc
- [sui-move-defi-lending](https://github.com/Proxtox/sui-move-defi-lending) — Move-based DeFi lending on Sui
- [base-defi-perpetuals-example](https://github.com/Proxtox/base-defi-perpetuals-example) — On-chain perpetuals trading primitive
- [bsc-yield-aggregator-vaults](https://github.com/Proxtox/bsc-yield-aggregator-vaults) — Auto-compounding yield vaults on BSC

## Stablecoins, Payments & Treasury
- [arc-usdc-payment-processor](https://github.com/Proxtox/arc-usdc-payment-processor) — USDC payment processing on Arc
- [arc-stablecoin-settlement-layer](https://github.com/Proxtox/arc-stablecoin-settlement-layer) — High-throughput settlement rails on Arc
- [arc-institutional-treasury](https://github.com/Proxtox/arc-institutional-treasury) — Multi-sig + scheduled payments for institutions

## NFTs, Gaming & Rewards
- [solana-nft-rental-marketplace](https://github.com/Proxtox/solana-nft-rental-marketplace) — NFT rental/leasing dApp on Solana
- [solana-gamefi-rewards-system](https://github.com/Proxtox/solana-gamefi-rewards-system) — Efficient GameFi rewards on Solana

## Meme Coins & Token Launches
- [base-meme-coin-launchpad](https://github.com/Proxtox/base-meme-coin-launchpad) — Fair-launch meme coin platform on Base

## RWA Tokenization & Credentials

- [solana-stablecoin-dex](https://github.com/Proxtox/solana-stablecoin-dex) — High-performance stablecoin AMM on Solana
- [ethereum-l2-base-comparison](https://github.com/Proxtox/ethereum-l2-base-comparison) — Educational L2 deployment & benchmarking comparison

- Smart contracts (Solidity / Move / Anchor)
- Deployment scripts
- Unit & integration tests
- GitHub Actions CI workflows
- Frontend examples (React / Next.js where applicable)
- Rich documentation
