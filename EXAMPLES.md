# Example Use Cases

Ideas for what you can build with these starters.

## Cross-Chain Applications
- Multi-chain stablecoin payment app
- Cross-chain yield aggregator
- Unified dashboard for positions across chains

## DeFi
- Custom lending protocol on Arc or Sui
- Perpetuals trading interface
- Yield vault with auto-compounding

## NFTs & Gaming
- NFT rental marketplace
- GameFi reward distribution system
- On-chain achievements and credentials

## Token Launches
- Fair meme coin launchpad
- RWA tokenization platform

## Tools
- Multi-chain portfolio tracker
- Liquidity router with best-route finding
- Institutional treasury management tool

Mix and match starters to create more complex applications!
