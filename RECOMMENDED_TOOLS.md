# Recommended Tools & Resources

Helpful tools when working with these starters.

## Development
- Hardhat (EVM)
- Anchor (Solana)
- Sui CLI & Move
- Foundry (fast EVM testing)

## Frontend
- Next.js + Tailwind
- viem + wagmi (EVM)
- @solana/wallet-adapter (Solana)
- RainbowKit or Privy (wallet connection)

## Cross-Chain
- Circle CCTP
- LayerZero
- Axelar
- Wormhole

## Oracles & Data
- Pyth Network (excellent on many chains)
- Chainlink

## Testing & CI
- Hardhat test framework
- GitHub Actions (already included in many repos)

## Useful Resources
- Base Docs
- Arc Docs
- Solana Docs
- Sui Docs

These tools pair well with the starters in this collection.
