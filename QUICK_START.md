# Quick Start Guide

Get up and running with the collection in minutes.

## 1. Find the Right Starter
Browse the [SHOWCASE.md](./SHOWCASE.md) and pick a repo that matches your goal.

## 2. Clone and Install
```bash
git clone https://github.com/Proxtox/<repo-name>.git
cd <repo-name>
npm install          # for EVM repos
# or
anchor build        # for Solana repos
sui move build      # for Sui repos
```

## 3. Deploy Locally
Most repos include a `scripts/deploy.js` or equivalent. Run it on the appropriate testnet.

## 4. Explore the Code
Start with the main contract/program file and the README.

## 5. Customize
Modify the code to fit your use case, add your frontend, and deploy to mainnet when ready.

Need inspiration? Check the INTEGRATION_GUIDE.md in many repos for ideas on how to extend them.
