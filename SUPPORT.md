# Support

## Getting Help

- Check the individual repo's README and INTEGRATION_GUIDE.md first.
- Browse the [FAQ.md](./FAQ.md) and [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) in this showcase.
- Open an issue in the relevant repository if you find a bug or have a feature request.

## Community

- Star the repos you find useful.
- Share your projects built with these starters.
- Contribute improvements via pull requests (see CONTRIBUTING.md).

We're building this collection to help the Web3 developer community. Your feedback and contributions are appreciated!
