# Contributing Guidelines

Thank you for considering contributing to this collection!

## Ways to Contribute
- Improve documentation and guides
- Add tests or fix bugs in existing starters
- Enhance frontend examples
- Suggest or create new high-quality starters
- Improve the showcase repo itself

## Pull Request Process
1. Fork the relevant repository
2. Create a feature branch
3. Make your changes with clear commit messages
4. Ensure tests pass (where applicable)
5. Submit a pull request with a clear description

We appreciate all contributions that help make these starters better for the community!
