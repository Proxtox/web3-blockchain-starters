# Versioning

This collection follows semantic versioning principles where applicable.

## Current Status
- Most individual starters are at v0.1.0 or early versions.
- The showcase and supporting files are actively being improved.

## Updating Starters
When improving a starter:
- Update the version in package.json or relevant files if applicable.
- Document changes in the repo's CHANGELOG.md (if present).
- Consider updating the main showcase CHANGELOG.md.

We aim for backward compatibility where possible while continuously improving quality.
