# Frequently Asked Questions

## General
**Q: What is this collection for?**
A: It provides ready-to-use starter repositories to help developers quickly build on Base, Arc, Solana, Ethereum, Sui, and BSC.

**Q: Are these production-ready?**
A: They are strong starting points with good structure, tests, and documentation. Always audit and test thoroughly before mainnet use.

## Using the Starters
**Q: Which repo should I start with?**
A: Check [SHOWCASE.md](./SHOWCASE.md) and pick based on your goal (e.g., cross-chain, lending, NFTs).

**Q: Can I combine multiple starters?**
A: Yes! Many are designed to work well together (e.g., using the multi-chain toolkit with a lending pool).

## Contributing
**Q: How can I contribute?**
A: See [CONTRIBUTING.md](./CONTRIBUTING.md). Improvements to documentation, tests, frontends, or new starter ideas are all welcome.

## Support
**Q: Where can I get help?**
A: Open an issue in the relevant repository or in the showcase repo.
