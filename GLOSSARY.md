# Glossary

Common terms used across this collection.

- **CCTP**: Circle Cross-Chain Transfer Protocol — used for fast, secure USDC transfers between chains.
- **RWA**: Real World Asset — tokenized versions of traditional assets (real estate, bonds, etc.).
- **AMM**: Automated Market Maker — the mechanism behind most DEXes.
- **Anchor**: Solana's framework for writing smart contracts in Rust.
- **Move**: The programming language used by Sui (and Aptos).
- **Perpetuals**: Perpetual futures contracts — derivatives without expiration.
- **Soulbound**: Non-transferable tokens/credentials (often used for identity or achievements).

This list will grow as the collection expands.
