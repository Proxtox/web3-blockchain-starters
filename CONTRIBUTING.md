# Contributing to Web3 & Blockchain Starters

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository you want to improve
2. **Create a branch** for your changes
3. **Make your changes** (bug fixes, new features, better docs, etc.)
4. **Test** your changes locally
5. **Submit a Pull Request**

## What We're Looking For

- Improvements to existing starters
- New high-quality starter ideas
- Better documentation and examples
- Frontend enhancements
- Additional test coverage
- CI/CD improvements

## Code Style

- Follow the existing structure in each repo
- Write clear commit messages
- Add tests when adding new functionality

## Questions?

Feel free to open an issue in any of the repositories.

Happy building! 🚀
